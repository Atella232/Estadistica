Conceptos Clave: Visualizando los Datos

  • Introducción
  • Diagrama de Barras
  • Diagrama de Sectores

Las tablas de frecuencias organizan los datos, pero a menudo una imagen vale más que mil números. Los gráficos estadísticos nos permiten visualizar la información de las tablas de forma rápida y atractiva, facilitando la comparación de datos y la identificación de patrones a simple vista. Para 1º de ESO, nos centraremos en dos tipos fundamentales:

La elección entre un diagrama de barras y uno de sectores depende de qué queramos destacar. Si queremos comparar las frecuencias exactas entre categorías, el de barras suele ser más claro. Si queremos enfatizar la contribución de cada categoría al total, el de sectores es más visual. Es fundamental entender que ambos gráficos son representaciones visuales de la información contenida en la tabla de frecuencias; sin una tabla correcta, el gráfico será incorrecto.

Diagrama de Barras

Uso Principal: Ideal para representar variables cualitativas o cuantitativas discretas.

Construcción:
  • Eje Horizontal (X): Se colocan los valores o categorías de la variable.
  • Eje Vertical (Y): Se representa la frecuencia (absoluta $f_i$ o relativa $h_i$) con una escala adecuada.
  • Barras Rectangulares: Se dibuja una barra rectangular sobre cada valor/categoría del eje X. La altura de cada barra es proporcional a su frecuencia.
  • ¡Importante! Separación: Las barras deben estar separadas entre sí para indicar que los valores son distintos o las categorías son independientes.
Interpretación:

Permite comparar fácilmente las frecuencias de las distintas categorías (la barra más alta es la más frecuente, etc.).

Diagrama de Sectores (Circular o de Tarta)

Uso Principal: Muy útil para representar variables cualitativas o cuantitativas discretas cuando queremos mostrar cómo se reparte un total ($N$) entre las diferentes categorías o valores. Es decir, para visualizar la proporción o porcentaje de cada parte respecto al todo.

Construcción:
  • Círculo Completo: Representa el 100% de los datos o el total $N$.
  • Sectores: El círculo se divide en sectores (como porciones de una tarta). Cada sector corresponde a un valor o categoría de la variable.
  • Ángulo del Sector: El tamaño (ángulo central) de cada sector es proporcional a su frecuencia (absoluta o relativa). Para calcular el ángulo de cada sector:
  • Ángulo $= h_i \times 360^\circ = \frac{f_i}{N} \times 360^\circ$
  • Se necesita un transportador de ángulos para dibujarlo con precisión. Es recomendable incluir una leyenda o etiquetar cada sector con su categoría y/o porcentaje.
Interpretación:

Permite ver rápidamente qué categoría representa la mayor o menor parte del total.
